# SNS怪談・4要素分析プロトタイプ→[demo](https://wasuke.github.io/folklore/)
## 概要

本システムは、SNS上の怪談投稿を対象に、以下の4要素を推定する実験的分析ツールである。

・実話性\
・匿名性\
・共感性\
・参加性

さらに、媒体特性（プラットフォームごとの重み）と反応量を統合し、「バズ推定値」を算出する。

本システムは三層分離構造を持つ。

・data.json ＝ 投稿データ\
・lex.json ＝ 語彙理論モデル\
・platforms.json ＝ 媒体特性モデル\
・index.html ＝ 分析エンジンおよび可視化UI

理論と実装を分離した研究用プロトタイプである。

## 実行方法

### 重要

ブラウザで直接開く（file:///）と fetch() が動作しない。\
必ずローカルサーバー経由で実行する。

### 簡易サーバー起動例（Python）

``` bash
python -m http.server 8000
```

ブラウザで以下を開く。

    http://localhost:8000

## data.json 形式

投稿オブジェクトの配列。

``` json
[
  {
    "id": "p1",
    "platform": "TikTok",
    "author": "匿名",
    "postedAt": "2026-02-08T23:12:00",
    "likes": 18400,
    "shares": 920,
    "comments": 1400,
    "text": "昨日の深夜。\n某県のトンネルで動画を撮った。"
  }
]
```

### 必須項目

・id\
・text

### 推奨項目

・platform\
・postedAt\
・likes\
・shares\
・comments

改行は `\n` を使用する。

------------------------------------------------------------------------

## lex.json 形式

4要素ごとの語彙と重み。

``` json
{
  "real": [
    { "w": ["実話", "証拠"], "p": 0.10 }
  ],
  "anon": [
    { "w": ["匿名", "某所"], "p": 0.12 }
  ],
  "empa": [
    { "w": ["怖い", "眠れない"], "p": 0.10 }
  ],
  "part": [
    { "w": ["検証", "考察"], "p": 0.10 }
  ]
}
```

・w ＝ 語彙配列\
・p ＝ 重み

------------------------------------------------------------------------

## platforms.json 形式

媒体特性の重み。

``` json
{
  "TikTok": {
    "engagementWeight": 0.40,
    "fourFactorWeight": 0.60,
    "description": "短尺動画・参加型拡散"
  },
  "default": {
    "engagementWeight": 0.30,
    "fourFactorWeight": 0.70,
    "description": "未定義媒体"
  }
}
```

バズ推定は次式で計算される。

    バズ推定 =
      (4要素平均 × fourFactorWeight) +
      (反応量 × engagementWeight)

------------------------------------------------------------------------

## 出力

・各投稿の4要素スコア\
・反応量\
・バズ推定値\
・根拠語彙

解析結果はJSON形式でエクスポート可能。

------------------------------------------------------------------------

## 設計思想

怪談拡散は次の式で近似される。

    拡散 = ナラティブ特性 × 媒体特性 × 反応量

語彙辞書と媒体重みは変更可能であり、検証可能な構造になっている。

------------------------------------------------------------------------

## 注意事項

本ツールは研究用の簡易モデルであり、真偽判定や誹謗中傷の拡散を目的としない。\
データ取得にあたっては各媒体の規約と倫理的配慮を遵守すること。
